# This is a school project about the unit circle

## Steps to install
1- Make sure you have Nodejs Installed

2- Run:  `npm i`

3- Run: `npm run build`

3- Run: `npm run preview`

4-Open http://locahost:4173/


## Description

This is an interactive unit circle model that let's you test out different angles and get a point on the circle out of it
